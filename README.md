# FFXIV Mount Tracker

A personal hobby project built with React and Vite that allows users to browse and filter mounts from Final Fantasy XIV.

The app uses the public FFXIV Collect API to display:
- Mount images
- Acquisition sources
- Expansion information
- Ownership statistics

Users can filter mounts by:
- Source type (Trial, Raid, Quest, PvP, etc.)
- Expansion (ARR → Dawntrail)

---

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- FFXIV Collect API

---

## Features

- Dynamic mount fetching from API
- Interactive filter sidebar
- Expansion filtering
- Responsive layout
- Sticky filter panel for easier navigation
- Source icons and expansion icons
- Live API-driven data
- Direct (and indirect) links to sources
- Visualization with more information when clicking on a card
- Character integration
